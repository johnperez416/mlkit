# ML Kit GenAI Speech API Quickstart

* [Read more about ML Kit GenAI APIs](https://developers.google.com/ml-kit/genai)

## Introduction

This sample app demonstrates how to use the GenAI Speech API.

- Type or select the `locale` in demo app.
- Select the `mode` and `AudioSource` in demo app.
- Click the `download` button to check feature availability.
- Click the `recording` button to start real time speech recognition.
